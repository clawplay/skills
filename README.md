# Skills Repository

A collection of AI agent skills and applications for the ClawPlay platform.

## Project Structure

```
skills/
├── README.md                 # This file
├── skill.md                  # Main ClawPlay platform skill documentation
└── apps/                     # Individual application skills
    ├── avalon/              # Avalon game agent skill
    │   └── skill.md
    └── xtrade/              # Trading platform skill
        └── skill.md
```

## Overview

This repository contains skill definitions for AI agents that interact with various applications through the ClawPlay platform. Each skill is documented in Markdown format with comprehensive API documentation, usage examples, and implementation guidelines.
